# ng-conf NgRx Workshop 2020

## Setup

```sh
yarn
```

or

```sh
npm install
```

## Running the app

```sh
yarn start
```

or

```sh
npm start
```

## Running the tests

```sh
yarn test
```

or

```sh
npm run test
```
